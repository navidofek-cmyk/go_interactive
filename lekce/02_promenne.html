Lekce 02: 📦 Krabičky — proměnné a typy

📄 02_promenne.go ⭐ Velmi snadné ⏱ ~20 minut

🤔 Co se naučíš?

Proměnná je jako krabička s nálepkou. Na nálepce je jméno krabičky, uvnitř je hodnota. Krabičky mohou být různé druhy: pro celá čísla (int), pro text (string), pro pravda/lež (bool), pro desetinná čísla (float64).

Naučíš se vytvářet krabičky, měnit jejich obsah a počítat s čísly. Super! 🎉

💡 Představ si šuplíky ve svém pokoji. Jeden šuplík má nálepku "hračky", druhý "knížky". Proměnná funguje stejně — má jméno a uvnitř uchováváme hodnotu. Kdykoli chceš hodnotu změnit, prostě otevřeš šuplík a vyměníš obsah!
// Soubor: 02_promenne.go
// Spusť: go run 02_promenne.go

package main

import "fmt"

func main() {
    // === Krabičky (proměnné) ===

    // var = vytvoříme krabičku a napíšeme typ
    var jmeno string = "Tomáš"          // string = text
    var vek int = 10                    // int = celé číslo
    var oblibeneZvire string = "pes 🐕"
    var umimPlavat bool = true          // bool = pravda nebo lež

    fmt.Println("=== Moje krabičky ===")
    fmt.Println("Jméno:", jmeno)
    fmt.Println("Věk:", vek)
    fmt.Println("Oblíbené zvíře:", oblibeneZvire)
    fmt.Println("Umím plavat:", umimPlavat)
    fmt.Println()

    // Kratší způsob s :=
    // Go sám pozná, jaký typ krabičky potřebuješ!
    oblibenaHra := "Minecraft"   // string
    body := 150                  // int
    teplota := 36.6              // float64 (desetinné číslo)

    fmt.Println("=== Moje hry a skóre ===")
    fmt.Println("Oblíbená hra:", oblibenaHra)
    fmt.Println("Moje body:", body)
    fmt.Printf("Tělesná teplota: %.1f °C\n", teplota)
    fmt.Println()

    // Krabičku můžeme změnit!
    fmt.Println("Původní body:", body)
    body = body + 50
    fmt.Println("Po bonusu:", body)
    body = body * 2
    fmt.Println("Po zdvojnásobení:", body)
    fmt.Println()

    // Různé výpisy
    fmt.Println("=== Počítáme ===")
    fmt.Printf("%s má %d let a sbírá %d her\n", jmeno, vek, 42)
    fmt.Println("Příští rok bude mít", vek+1, "let")
}
📺 Co program vypíše:
=== Moje krabičky === Jméno: Tomáš Věk: 10 Oblíbené zvíře: pes 🐕 Umím plavat: true === Moje hry a skóre === Oblíbená hra: Minecraft Moje body: 150 Tělesná teplota: 36.6 °C Původní body: 150 Po bonusu: 200 Po zdvojnásobení: 400 === Počítáme === Tomáš má 10 let a sbírá 42 her Příští rok bude mít 11 let
