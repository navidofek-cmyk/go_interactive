Lekce 05: 📋 Seznamy — slice a map

📄 05_slice_map.go ⭐⭐ Snadné ⏱ ~25 minut

🤔 Co se naučíš?

Slice je jako nákupní lístek — seznam věcí v řadě. Každá věc má své číslo (0, 1, 2...). Map je jako telefonní seznam — k jménu najdeš číslo, nebo k slovu najdeš překlad. Obě slouží k ukládání více věcí najednou.

💡 Slice = nákupní lístek (1. jablko, 2. mléko, 3. chléb — pořadí záleží!). Map = slovník angličtina-čeština (pes → dog, kočka → cat — hledáš podle slova).
// Soubor: 05_slice_map.go
// Spusť: go run 05_slice_map.go

package main

import "fmt"

func main() {
    // === SLICE — nákupní lístek ===
    fmt.Println("=== Moje oblíbená zvířata ===")

    zvirata := []string{"🐕 pes", "🐈 kočka", "🐇 králík", "🐹 křeček"}

    // Vypíšeme vše
    for i, zvire := range zvirata {
        fmt.Printf("%d. %s\n", i+1, zvire)
    }

    // Přidáme nové zvíře
    zvirata = append(zvirata, "🦜 papoušek")
    fmt.Printf("\nPřidali jsme papoušek! Teď jich máme: %d\n", len(zvirata))
    fmt.Println()

    // === Čísla v seznamu ===
    fmt.Println("=== Body ve hře ===")
    body := []int{150, 200, 75, 300, 125}

    celkem := 0
    for _, b := range body {
        celkem += b
        fmt.Printf("  kolo: %3d bodů\n", b)
    }
    fmt.Printf("Celkem: %d bodů\n", celkem)
    fmt.Println()

    // === MAP — slovník / telefonní seznam ===
    fmt.Println("=== Zvuky zvířat (slovník) ===")

    zvuky := map[string]string{
        "pes":   "Haf! 🐕",
        "kočka": "Mňau! 🐈",
        "kráva": "Mů! 🐄",
        "ovce":  "Bé! 🐑",
        "had":   "Ssss! 🐍",
    }

    // Vypíšeme celý slovník
    for zvire, zvuk := range zvuky {
        fmt.Printf("%-8s → %s\n", zvire, zvuk)
    }
    fmt.Println()

    // Hledáme konkrétní zvíře
    hledam := "kočka"
    zvuk, existuje := zvuky[hledam]
    if existuje {
        fmt.Printf("Zvuk pro %s: %s\n", hledam, zvuk)
    } else {
        fmt.Println("Zvíře nenalezeno!")
    }

    fmt.Println("\nSkvělé! Umíš pracovat se seznamy! 🎉")
}
📺 Co program vypíše:
=== Moje oblíbená zvířata === 1. 🐕 pes 2. 🐈 kočka 3. 🐇 králík 4. 🐹 křeček Přidali jsme papoušek! Teď jich máme: 5 === Body ve hře === kolo: 150 bodů kolo: 200 bodů kolo: 75 bodů kolo: 300 bodů kolo: 125 bodů Celkem: 850 bodů === Zvuky zvířat (slovník) === pes → Haf! 🐕 kočka → Mňau! 🐈 kráva → Mů! 🐄 ovce → Bé! 🐑 had → Ssss! 🐍 (Poznámka: pořadí v mapě se může lišit!) Zvuk pro kočka: Mňau! 🐈 Skvělé! Umíš pracovat se seznamy! 🎉

✏️ Zkus to sám!

  1. Přidej do slice zvirata své oblíbené zvíře pomocí append.
    🔍 Zobrazit nápovědu
    zvirata = append(zvirata, "🦊 liška")
  2. Přidej do mapy zvuky zvuk pro "žábu" — žába říká "Kvak!".
    🔍 Zobrazit nápovědu
    zvuky["žába"] = "Kvak! 🐸"
  3. Spočítej průměrný počet bodů z pole body a vypiš ho.
    🔍 Zobrazit nápovědu
    fmt.Printf("Průměr: %d bodů\n", celkem/len(body))
