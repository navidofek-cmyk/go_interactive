Lekce 17: 🏙️ Microservices — malé spolupracující servery

📄 3 soubory · 3 terminály ⭐⭐⭐⭐⭐ Mistrovská výzva! ⏱ ~50 minut

🤔 Co jsou microservices?

Velký program se dá rozdělit na malé, samostatné servery — každý dělá jen jednu věc. Navzájem si volají přes HTTP, jako kdyby si posílaly zprávy.

Tak funguje třeba Netflix: jeden server hledá filmy, druhý řeší platby, třetí doporučuje co sledovat — každý je samostatný program!

💡 Analogie — restaurace: Šéfkuchař nedělá všechno sám. Má kuchaře na pizzu (port 8081), kuchaře na dezerty (port 8082) a číšníka (port 8080) který objednávky přerozděluje. Každý je "microservice" — malá, samostatná služba.
Jak naše microservices spolu mluví:
Prohlížeč / curl ↓ HTTP GET :8080 ┌────────────────────┐ │ 🌐 GATEWAY │ port 8080 ← hlavní vstupní bod │ "Číšník" │ └────────┬───────────┘ │ volá ostatní služby ┌──────┴──────┐ ↓ ↓ ┌──────────┐ ┌──────────┐ │ 🍕 MENU │ │ 🍰 DEZERT│ │ port 8081│ │ port 8082│ └──────────┘ └──────────┘ vrátí JSON vrátí JSON
🍕 Služba 1: Menu server (port 8081)

Spusť v Terminálu 1: go run sluzba_menu.go

📄 sluzba_menu.go
// Soubor: sluzba_menu.go
// Spusť v Terminálu 1: go run sluzba_menu.go

package main

import (
	"encoding/json"
	"fmt"
	"net/http"
)

// Jidlo popisuje položku na menu
type Jidlo struct {
	Nazev string `json:"nazev"`
	Cena  int    `json:"cena"`
	Emoji string `json:"emoji"`
}

// Naše menu
var menu = []Jidlo{
	{Nazev: "Margherita", Cena: 180, Emoji: "🍕"},
	{Nazev: "Hawai",      Cena: 200, Emoji: "🍕"},
	{Nazev: "Bolognese",  Cena: 160, Emoji: "🍝"},
	{Nazev: "Caesar",     Cena: 140, Emoji: "🥗"},
}

func menuHandler(w http.ResponseWriter, r *http.Request) {
	fmt.Println("📋 Někdo si žádá menu!")
	w.Header().Set("Content-Type", "application/json")
	json.NewEncoder(w).Encode(menu)
}

func main() {
	http.HandleFunc("/menu", menuHandler)
	fmt.Println("🍕 Menu server běží na http://localhost:8081")
	fmt.Println("   Zkus: curl http://localhost:8081/menu")
	http.ListenAndServe(":8081", nil)
}

🌐 Služba 3: Gateway — číšník (port 8080)

Spusť v Terminálu 3: go run gateway.go — pak otevři http://localhost:8080

📄 gateway.go
// Soubor: gateway.go
// Spusť v Terminálu 3: go run gateway.go
// Pak otevři: http://localhost:8080

package main

import (
	"encoding/json"
	"fmt"
	"io"
	"net/http"
)

// Zavolá jinou microservice a vrátí odpověď jako text
func zavolajSluzbu(url string) string {
	resp, err := http.Get(url)
	if err != nil {
		return fmt.Sprintf(`"chyba: %v"`, err)
	}
	defer resp.Body.Close()
	data, _ := io.ReadAll(resp.Body)
	return string(data)
}

func hlavniHandler(w http.ResponseWriter, r *http.Request) {
	fmt.Println("🌐 Gateway: přišel požadavek, ptám se na služby...")

	// Zavoláme obě služby paralelně!
	menuCh := make(chan string, 1)
	dezertCh := make(chan string, 1)

	go func() { menuCh <- zavolajSluzbu("http://localhost:8081/menu") }()
	go func() { dezertCh <- zavolajSluzbu("http://localhost:8082/dezerty") }()

	menuData := <-menuCh
	dezertData := <-dezertCh

	// Vrátíme HTML stránku zákazníkovi
	w.Header().Set("Content-Type", "text/html; charset=utf-8")
	fmt.Fprintf(w, `<!DOCTYPE html>
<html><head><meta charset="utf-8">
<style>
  body { font-family: Arial; background: #0d1117; color: #e6edf3;
         max-width: 800px; margin: 2rem auto; padding: 1rem; }
  h1 { color: #58a6ff; }
  h2 { color: #ffa657; margin-top: 2rem; }
  pre { background: #161b22; padding: 1rem; border-radius: 8px;
        overflow-x: auto; color: #3fb950; font-size: .9rem; }
  .badge { background: #1f6feb; padding: .2rem .6rem; border-radius: 999px;
           font-size: .75rem; margin-left: .5rem; }
</style></head>
<body>
  <h1>🍽️ Restaurace Go <span class="badge">microservices</span></h1>
  <p>Gateway zavolala 2 microservices a spojila výsledky:</p>
  <h2>🍕 Menu (ze služby :8081)</h2>
  <pre>%s</pre>
  <h2>🍰 Dezerty (ze služby :8082)</h2>
  <pre>%s</pre>
</body></html>`, menuData, dezertData)

	fmt.Println("✅ Gateway: odpověď odeslána!")
}

func main() {
	http.HandleFunc("/", hlavniHandler)
	fmt.Println("🌐 Gateway běží na http://localhost:8080")
	fmt.Println("   Otevři prohlížeč na: http://localhost:8080")
	http.ListenAndServe(":8080", nil)
}
💡 Proč volat služby paralelně? Gateway spouští obě HTTP volání jako goroutiny najednou — menu i dezerty se načítají současně. Pokud by každé trvalo 100ms, celkem zaplatíme 100ms (ne 200ms). Tohle je v reálných systémech obrovský rozdíl!
🚀 Jak to celé spustit
Otevři 3 terminály a spusť v každém jeden příkaz:
Terminál 1: go run sluzba_menu.go → 🍕 Menu server běží na http://localhost:8081 Terminál 2: go run sluzba_dezerty.go → 🍰 Dezert server běží na http://localhost:8082 Terminál 3: go run gateway.go → 🌐 Gateway běží na http://localhost:8080 Prohlížeč: http://localhost:8080 → 🍽️ Uvidíš stránku se spojeným menu i dezerty!
💡 Vyzkoušej také curl:
curl http://localhost:8081/menu
curl http://localhost:8082/dezerty
curl http://localhost:8080
