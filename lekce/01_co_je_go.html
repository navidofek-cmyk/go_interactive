Lekce 01: 🐹 Co je Go? Ahoj, robote!

📄 01_co_je_go.go ⭐ Velmi snadné ⏱ ~15 minut

🤔 Co se naučíš?

Go je jazyk, kterým mluvíme s počítačem. Počítač je jako robot — neumí přemýšlet sám, ale dělá PŘESNĚ to, co mu řekneme. Každý Go program musí začít s package main a mít funkci main() — to je místo, kde program začne.

Naučíš se vypisovat text, počítat čísla a napsat svůj první program! 🎉

💡 Představ si, že počítač je robot. Program v Go je seznam příkazů, které robot dostane. Třeba: "Řekni Ahoj!", "Vypiš moje jméno", "Spočítej 5+3". Robot dělá příkazy jeden po druhém, od shora dolů.
// Soubor: 01_co_je_go.go
// Spusť: go run 01_co_je_go.go

package main   // každý Go program musí mít toto!

import "fmt"   // fmt nám dává příkaz Println

func main() {
    // main() je začátek každého programu
    // Vše uvnitř { } se spustí od shora dolů

    fmt.Println("Ahoj! Jsem tvůj první Go program! 🐹")
    fmt.Println("Jmenuji se Robot Go.")
    fmt.Println()  // prázdný řádek

    fmt.Println("=== Dnes se naučím: ===")
    fmt.Println("1. Vypisovat text")
    fmt.Println("2. Počítat čísla")
    fmt.Println("3. Mluvit s robotem")
    fmt.Println()

    // Počítáme jako kalkulačka!
    fmt.Println("=== Počítám za tebe: ===")
    fmt.Println("5 + 3 =", 5+3)
    fmt.Println("10 - 4 =", 10-4)
    fmt.Println("6 * 7 =", 6*7)
    fmt.Println("15 / 3 =", 15/3)
    fmt.Println()

    // fmt.Printf umí formátovat text
    fmt.Printf("Mám %d let a jmenuji se %s!\n", 10, "Robot Go")
    fmt.Printf("Moje oblíbená barva je %s 🎨\n", "modrá")
    fmt.Println()

    fmt.Println("Skvělé! Napsal jsi svůj první program! 🎉")
}
📺 Co program vypíše:
Ahoj! Jsem tvůj první Go program! 🐹 Jmenuji se Robot Go. === Dnes se naučím: === 1. Vypisovat text 2. Počítat čísla 3. Mluvit s robotem === Počítám za tebe: === 5 + 3 = 8 10 - 4 = 6 6 * 7 = 42 15 / 3 = 5 Mám 10 let a jmenuji se Robot Go! Moje oblíbená barva je modrá 🎨 Skvělé! Napsal jsi svůj první program! 🎉

✏️ Zkus to sám!

  1. Změň text "Robot Go" na své vlastní jméno a znovu spusť program.
    🔍 Zobrazit nápovědu
    fmt.Printf("Mám %d let a jmenuji se %s!\n", 10, "Tvoje jméno")
  2. Přidej výpočet 99 * 99 a vypiš výsledek.
    🔍 Zobrazit nápovědu
    fmt.Println("99 * 99 =", 99*99)
  3. Vypiš svou oblíbenou barvu pomocí fmt.Printf.
    🔍 Zobrazit nápovědu
    fmt.Printf("Moje oblíbená barva je %s\n", "červená")
