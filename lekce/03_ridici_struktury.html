Lekce 03: 🤔 Rozhodování — if, for, switch

📄 03_ridici_struktury.go ⭐⭐ Snadné ⏱ ~25 minut

🤔 Co se naučíš?

Počítač se umí rozhodovat! Pomocí if řekneme: "Jestli je to pravda, udělej tohle. Jinak udělej tamto." Pomocí for věci opakujeme — třeba 10× vypiš hvězdičku. A switch je jako výběr z menu — vyber si jednu možnost.

💡 if = stopka na silnici (červená = stůj, zelená = jdi). for = robot, kterému řekneš "Udělej dřep 10×". switch = výběr zmrzliny — jahoda, čokoláda nebo vanilka?
// Soubor: 03_ridici_struktury.go
// Spusť: go run 03_ridici_struktury.go

package main

import "fmt"

func main() {
    // === IF — Rozhodování ===
    fmt.Println("=== Stopka ===")

    vek := 10

    if vek >= 18 {
        fmt.Println("Smíš jet autem 🚗")
    } else if vek >= 15 {
        fmt.Println("Smíš jezdit na mopedku 🛵")
    } else {
        fmt.Println("Jeď na kole! 🚲")
    }
    fmt.Println()

    // === FOR — Opakování ===
    fmt.Println("=== Kreslíme hvězdičky ===")

    // Opakuj 5×
    for i := 1; i <= 5; i++ {
        fmt.Print("⭐")
    }
    fmt.Println()

    // Trojúhelník z hvězdiček
    fmt.Println("\n=== Trojúhelník ===")
    for radek := 1; radek <= 5; radek++ {
        for hvezdicka := 0; hvezdicka < radek; hvezdicka++ {
            fmt.Print("⭐")
        }
        fmt.Println()
    }

    // === SWITCH — Výběr z menu ===
    fmt.Println("\n=== Zvuky zvířat ===")

    zvirata := []string{"pes", "kočka", "kráva", "žába"}

    for _, zvire := range zvirata {
        switch zvire {
        case "pes":
            fmt.Println("🐕 Pes říká: Haf!")
        case "kočka":
            fmt.Println("🐈 Kočka říká: Mňau!")
        case "kráva":
            fmt.Println("🐄 Kráva říká: Mů!")
        default:
            fmt.Println("🐸 Neznámé zvíře říká: ...?")
        }
    }

    fmt.Println("\nSkvělé! Umíš rozhodovat! 🎉")
}
📺 Co program vypíše:
=== Stopka === Jeď na kole! 🚲 === Kreslíme hvězdičky === ⭐⭐⭐⭐⭐ === Trojúhelník === ⭐ ⭐⭐ ⭐⭐⭐ ⭐⭐⭐⭐ ⭐⭐⭐⭐⭐ === Zvuky zvířat === 🐕 Pes říká: Haf! 🐈 Kočka říká: Mňau! 🐄 Kráva říká: Mů! 🐸 Neznámé zvíře říká: ...? Skvělé! Umíš rozhodovat! 🎉
