🌐 Vlastní web — HTTP server

📄 12_http_server.go ⭐⭐⭐⭐⭐ Výzva! ⏱ ~40 minut

🤔 Co se naučíš?

HTTP server je program, který "obsluhuje" webové stránky. Když napíšeš adresu do prohlížeče, tvůj prohlížeč se zeptá serveru a server odpovědi. Go má server zabudovaný — stačí pár řádků! Toto je poslední lekce — jsi téměř Go programátor!
💡 HTTP server = recepce v hotelu. Přijdeš (pošleš požadavek), recepce tě obslouží (server odpoví). Bez recepce se do hotelu nedostaneš.
// Soubor: 12_http_server.go
// Spusť: go run 12_http_server.go
// Pak otevři prohlížeč na: http://localhost:8080

package main

import (
    "fmt"
    "net/http"
    "time"
)

// Hlavní stránka
func domovPage(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, `<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>Můj Go Web! 🐹</title>
<style>body{font-family:Arial;background:#0d1117;color:#e6edf3;text-align:center;padding:2rem;}
h1{color:#58a6ff;font-size:3rem;} p{font-size:1.3rem;}</style></head>
<body>
<h1>🐹 Ahoj! Jsem Go Server!</h1>
<p>Tento web běží díky Go programu který jsi napsal!</p>
<p><a href="/cas" style="color:#58a6ff">Kolik je hodin? ⏰</a></p>
<p><a href="/ahoj?jmeno=Tomas" style="color:#3fb950">Pozdrav mě! 👋</a></p>
</body></html>`)
}

// Stránka s časem
func casPage(w http.ResponseWriter, r *http.Request) {
    cas := time.Now().Format("15:04:05")
    datum := time.Now().Format("2. 1. 2006")
    fmt.Fprintf(w, "<h1 style='font-family:Arial;color:#3fb950'>⏰ Právě je %s</h1><p style='font-family:Arial'>Datum: %s</p>", cas, datum)
}

// Stránka s pozdravem
func ahojPage(w http.ResponseWriter, r *http.Request) {
    jmeno := r.URL.Query().Get("jmeno")
    if jmeno == "" {
        jmeno = "světe"
    }
    fmt.Fprintf(w, "<h1 style='font-family:Arial;color:#ffa657'>👋 Ahoj, %s!</h1><p style='font-family:Arial'>Jsi super programátor!</p>", jmeno)
}

func main() {
    // Nastavíme stránky
    http.HandleFunc("/", domovPage)
    http.HandleFunc("/cas", casPage)
    http.HandleFunc("/ahoj", ahojPage)

    port := ":8080"
    fmt.Println("🚀 Server spuštěn!")
    fmt.Println("🌐 Otevři prohlížeč na: http://localhost" + port)
    fmt.Println("📄 Stránky: /  /cas  /ahoj?jmeno=Tomas")
    fmt.Println("⛔ Pro ukončení stiskni Ctrl+C")

    // Spustíme server
    err := http.ListenAndServe(port, nil)
    if err != nil {
        fmt.Println("❌ Chyba serveru:", err)
    }
}
📺 Co program vypíše (v terminálu):
🚀 Server spuštěn! 🌐 Otevři prohlížeč na: http://localhost:8080 📄 Stránky: / /cas /ahoj?jmeno=Tomas ⛔ Pro ukončení stiskni Ctrl+C (Server běží a čeká — otevři prohlížeč na http://localhost:8080 a uvidíš svůj web! 🎉) (Zastav program klávesou Ctrl+C)

✏️ Zkus to sám!

  1. Přidej stránku "/barvy" která zobrazí seznam barev jako barevný HTML text.
    🔍 Zobrazit nápovědu
    http.HandleFunc("/barvy", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, `<h1 style='color:red'>Červená</h1>
<h1 style='color:blue'>Modrá</h1>
<h1 style='color:green'>Zelená</h1>`)
})
  2. Změň port z 8080 na 9000. Co musíš změnit v prohlížeči?
    🔍 Zobrazit nápovědu
    // Změň v kódu:
port := ":9000"

// V prohlížeči otevři:
// http://localhost:9000
  3. Přidej stránku "/moje-jmeno" která zobrazí tvoje jméno velkým písmem.
    🔍 Zobrazit nápovědu
    http.HandleFunc("/moje-jmeno", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "<h1 style='font-family:Arial;font-size:5rem'>Tomáš</h1>")
