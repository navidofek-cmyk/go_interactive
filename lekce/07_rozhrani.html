🦸 Superschopnosti — rozhraní

📄 07_rozhrani.go ⭐⭐⭐ Střední ⏱ ~30 minut

🤔 Co se naučíš?

Interface (rozhraní) je jako superschopnost. Třeba "umí létat" je superschopnost — ptáci, superhrdinové i letadla ji mají, každý jinak. V Go interface říká "tato věc musí umět tohle". Díky tomu můžeš psát funkce, které fungují s různými typy najednou!
💡 Interface = superschopnost. "Umí mluvit" — pes říká Haf, kočka Mňau, člověk říká Ahoj. Všichni "umí mluvit" ale každý jinak!
package main

import "fmt"

// Interface = seznam schopností
// Každé zvíře musí umět: Zvuk() a Popis()
type Zvire interface {
    Zvuk() string
    Popis() string
}

// Pes splňuje interface Zvire
type Pes struct {
    Jmeno string
}
func (p Pes) Zvuk() string  { return "Haf! 🐕" }
func (p Pes) Popis() string { return "Pes jménem " + p.Jmeno }

// Kočka splňuje interface Zvire
type Kocka struct {
    Barva string
}
func (k Kocka) Zvuk() string  { return "Mňau! 🐈" }
func (k Kocka) Popis() string { return k.Barva + " kočka" }

// Žába splňuje interface Zvire
type Zaba struct{}
func (z Zaba) Zvuk() string  { return "Kvak! 🐸" }
func (z Zaba) Popis() string { return "Zelená žába" }

// Tato funkce funguje s JAKÝMKOLIV zvířetem!
func predstavSeZvire(z Zvire) {
    fmt.Printf("  %s říká: %s\n", z.Popis(), z.Zvuk())
}

func main() {
    fmt.Println("=== Zvířecí zpěvník ===\n")

    // Různá zvířata v jednom seznamu!
    zvirata := []Zvire{
        Pes{Jmeno: "Rex"},
        Pes{Jmeno: "Bořek"},
        Kocka{Barva: "Černá"},
        Kocka{Barva: "Bílá"},
        Zaba{},
    }

    for _, z := range zvirata {
        predstavSeZvire(z)
    }

    fmt.Println("\n=== Kdo dělá jaký zvuk? ===")
    for _, z := range zvirata {
        fmt.Printf("  %s\n", z.Zvuk())
    }

    fmt.Println("\nSkvělé! Interfaces jsou jako superschopnosti! 🦸🎉")
}
📺 Co program vypíše:
=== Zvířecí zpěvník === Pes jménem Rex říká: Haf! 🐕 Pes jménem Bořek říká: Haf! 🐕 Černá kočka říká: Mňau! 🐈 Bílá kočka říká: Mňau! 🐈 Zelená žába říká: Kvak! 🐸 === Kdo dělá jaký zvuk? === Haf! 🐕 Haf! 🐕 Mňau! 🐈 Mňau! 🐈 Kvak! 🐸 Skvělé! Interfaces jsou jako superschopnosti! 🦸🎉
